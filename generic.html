About Me

passinate and destructive soul

Hello! This is Vicky. I'm a junior majoring in Computer Science at Cornell University.

Where do I start? I didn't start programming until college when I was taking my first programming class as a graduation requirement. I struggled quite a lot throughout that course, but it did bring a whole new world that was so different than other theoretical classes I had taken. The technology world always seemed like a fasinating treasure behind a locked door, and this intro class somehow easily unlooked it. It convinced me that I was able to build something complicated from zero in such short time; and if I set my mind on a task, I can build it. This belief enpowers me and pushes me going.
